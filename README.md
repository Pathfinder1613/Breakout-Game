Breakout Game
Features
Difficulty Levels

Choose from four difficulty settings:

Normal – Standard gameplay with 5 lives
Hard – Faster ball speed with 3 lives
Power Game – Includes power-ups with 3 lives
Insane – Extreme difficulty with fewer lives and shrinking paddle power-ups
Game Modes
Normal Mode

Standard Breakout gameplay.

Power-Up Mode

Play with special power-ups that can help or challenge the player.

Available power-ups:

Expand Paddle
Multiball
Slow Ball
Speed Up Paddle
Classic Mode

Traditional Breakout gameplay with no power-ups.

Power-Ups
Expand Paddle

Temporarily increases the width of the paddle, making it easier to hit the ball.

Shrink Paddle

Temporarily decreases the width of the paddle, making the game more challenging. (Insane Mode only)

Multiball

Spawns additional balls, allowing the player to break multiple bricks at once.

Slow Ball

Temporarily reduces the ball's speed, making it easier to control.

Speed Up Paddle

Temporarily increases paddle movement speed.

HUD (Heads-Up Display)

The in-game HUD displays:

Current Score
Remaining Lives
Current Round
Replay & Reset
Restart the current game at any time.
Return to the main menu and start a new game.
Reset progress and begin a fresh match.
